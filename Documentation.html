Student Expense Tracker

External link to source zip
The Student Expense Tracker app helps college students, who may not be adept with finance, keep
track of their spending. The app does this by allowing a user to set a limit on how much they wish to
spend, displaying previous period's data, and allows users to put their expenses into different
categories. These categories are; food,rent, health, other, and school. Upon clicking on one
of these options, the expense history for each is displayed. When a user selects the "Add Expense"
button, a page is displayed where the use may enter the date, category, and amount. An optional
description field is provided as well. Since this app is not directly connected to a bank, there is no
need for a login or even an internet connection. This not only makes it simpiler to use, but helps keep
hackers from stealing personal, banking, and financial information.
